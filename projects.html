Book Kiosk

The book kiosk is a java spring/hibernate application that is designed to replicate a vending machine, but for books essentially. Within it students can donate new books they own, search for books by author or title, borrow a book from the kiosk, or return a book in their possession to the kiosk. The librarian may additionally delete a book from the kisok, and view the amount of times each student has logged into the application. The application is built using spring security and authentication for purposes of logging in.

Customer Manager

The customer CRUD database application is designed to catalogue and retrieve a business's clientele information such as name, email, and phone number. The application consists of a simple landing page and another set of pages whhich carry out the CRUD operations. This application was made using Java, Spring, Hibernate, HTML, CSS, JavaScript, SQL, and Bootstrap framework.

The Division Gearset Bundler

An older project based on the videogame Tom Clancy’s The Division. Players did quest, to find equipment, to do harder quests requiring better equipment. The particular combination of equipment players chose was very important, but there was no way in-game to remember previous combinations or loadouts. Constructed using mostly JavaScript and a user’s web browser’s native storage, hosted on GitHub pages.Enabled players to construct, bundle, name, store, and recall specific item sets and combinations. Eventually the developers added their own in game equipment bundle module (much better looking than mine).
